### Postgres

Postgres version --latest
This is the database for persisiting information of `views`
On build a table `STORY_VIEWS` is created

```sql
CREATE TABLE STORY_VIEWS (
  id VARCHAR(50) PRIMARY KEY,
  views INT
);
```
